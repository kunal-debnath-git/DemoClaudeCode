# DemoClaudeCode

A simple static HTML project containing a welcome page with CSS animations and vanilla JavaScript effects.

## Files

- `welcome.html` - Welcome screen with animated gradient text, floating orbs, particle effects, and confetti celebration on button click
- `nayabad_gated_communities.html` - Nayabad gated communities page
- `Course_notes.md` - Course notes

## Development

No build tools or dependencies required. Open the HTML files directly in a browser to view.
